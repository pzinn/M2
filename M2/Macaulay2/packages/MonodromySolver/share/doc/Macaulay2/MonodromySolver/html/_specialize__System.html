MonodromySolver :: specializeSystem

specializeSystem -- specialize parametric system at a point in the parameter space.

Synopsis

  • Usage:
    equations = specializeSystem(p0, PS)
  • Inputs:
    • p0, a point, in parameter space
    • PS, a polynomial system, with parameters as coefficients
  • Outputs:
    • equations, a list, containing equtaions of system with parameters specialized at p0

Ways to use specializeSystem :

  • specializeSystem(Point,Matrix)
  • specializeSystem(Point,PolySystem)
