Description

This blackbox solver is similar in usage to sparseMonodromySolve, but with "technical" output.

i1 : R = CC[a,b,c,d][A,B]

o1 = R

o1 : PolynomialRing
i2 : polys = polySystem {A^2*a+B^2*b,A*B*c+d}

o2 = polys

o2 : PolySystem
i3 : setRandomSeed 0;
i4 : (V,npaths) = monodromySolve(polys, NumberOfNodes => 3);
i5 : peek V
